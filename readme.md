1. 이미지 데이터 전처리(Image Data Preprocessing)

 + 차량 내부, 외부 분류 by YOLOV3

2. 데이터셋 수정

  + 학습을 위한 차량 이미지 데이터가 깔끔하고 명확하지 않아서 데이터셋을 수정하기로 결정.

  + 추가로 각 차량 모델의 홈페이지에서 데이터를 확보.

  + 한 차량 모델에 대해 모델링 된 차량 이미지 120장, 크롤링을 통한 차량 이미지 180장으로 총 300장으로 재구성.
